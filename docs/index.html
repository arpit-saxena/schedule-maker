Schedule maker

Sem 1 2022-23 | IIT Delhi

Made by Arpit Saxena | Current calendar contributed by T Pratham | Find source code on GitHub

Specify course names and slot to generate an iCalendar file for your semester. It can be used to add your schedule to any calendar app of your choosing, though I have only tested it with Google Calendar. It will take into account all extra working days and holidays. It is advised to create a new calendar to import into so it can be deleted in case of an issue.

DISCLAIMER: It is advised to create a new calendar to import into so all imported events can be deleted in case of an issue. While I have tried to ensure correctness, there may be some errors in the schedule or slot timings. If you find any, please file an issue on the GitHub page and I'll fix it ASAP.

Info: For slot name, write slot name in all caps. For example: C or D or AA
Currently supported slots: A, B, C, D, E, F, H, J, K, L, M, AA, AB, AC, AD
New: Check the box before a course's name to enter custom timings and repeating days for it.

Steps to import on Google Calendar

  • Go to https://calendar.google.com on your PC
  • Click on gear icon on top right and click on settings
  • On left hand side menu, click on "Add calendar" and create new calendar. Choose an appropriate name for calendar and click "Create Calendar"
  • When calendar is created, click on "Import & Export" on left hand side menu
  • In the "Import" section, choose the file you downloaded from this website and select the calendar you created previously. Then click import

Known issues on other calendar clients

  • MacOS Client App does not add extra working days (e.g. Saturdays) properly
