Schedule maker

Sem 1 2020-21 | IIT Delhi

Made by Arpit Saxena | Find source code on GitHub

Specify course names and slot to generate an iCalendar file for your semester. It can be used to add your schedule to any calendar app of your choosing, though I have only tested it with Google Calendar. It will take into account all extra working days and holidays. It is advised to create a new calendar to import into so it can be deleted in case of an issue.

DISCLAIMER: It is advised to create a new calendar to import into so all imported events can be deleted in case of an issue. While I have tried to ensure correctness, there may be some errors in the schedule or slot timings. If you find any, please file an issue on the GitHub page and I'll fix it ASAP.
